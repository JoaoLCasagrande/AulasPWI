Aqui veremos um pouco sobre comandos da programação HTML e suas funções

Comando Funcionalidade Que parte da programação fica
HTML Diz ao navegador que o que estiver dentro dessa tag deve ser encarado como elementos de um código HTML body
body É o comando responsável por guardar tudo que é conteúdo do site body
title O título de uma página de um site é o que aparece no seu browser, na barra superior, chamada de title bar. O título é um texto que define, de forma resumida, o assunto que a página de seu site está lidando.
Cabeçalhos (h1,h2,h3,h4,h5 ou h6) Cabeçalhos podem ser usados em qualquer parte da página em HTML para dar destaque maior ou menor a algum termo do texto
Tipos de letras i- Itálico b- Bold code- Code
Listas ul- afasta o conteúdo para a direita li- coloca uma marca antes de cada item ol- cria uma lista ordenada
