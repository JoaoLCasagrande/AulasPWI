Aqui veremos um pouco sobre comandos da programação HTML e suas funções

Comando Funcionalidade Que parte da programação fica
HTML Diz ao navegador que o que estiver dentro dessa tag deve ser encarado como elementos de um código HTML Iníio da Programação
head É usada como cabeçalho, ou seja, um conjunto de informações de uma página de um site.Na criação de sites, a tag head tem uma função similar: a tag head serve para armazenar informações de uma página de seu site. Essas informações são invisíveis, ou seja, os usuários que estiverem navegando pelo seu website não verão o que existe na tag head. Início da Programação
body É o comando responsável por guardar tudo que é conteúdo do site. A maior parte das coisas que são escritas aqui são as visíveis para o usuário, é o conteúdo Após o Head
title O título de uma página de um site é o que aparece no seu browser, na barra superior, chamada de title bar. O título é um texto que define, de forma resumida, o assunto que a página de seu site está lidando. head
Cabeçalhos (h1,h2,h3,h4,h5 ou h6) Cabeçalhos podem ser usados em qualquer parte da página em HTML para dar destaque maior ou menor a algum termo do texto body
Tipos de letras (i-Itálico, b-Bold, code-Code e u-Sublinhado Alteram a fonte das letras que aparecerão no site body
Listas (ul,li e ol) ul- afasta o conteúdo para a direita
li- coloca uma marca antes de cada item
ol- cria uma lista ordenada		 body
Meta Charset A ideia é que com ele possamos representar qualquer caractere de qualquer idioma. Ao usar o charset UTF-8,por exemplo,dentro da tag meta de nosso arquivo estamos dizendo ao navegador: quando for ler esse arquivo, use as regras do charset UTF-8 head
Table Define que uma tabela será criada no seu site body
Border Esse comando é responsável por adicionar bordas em qualquer elemento html, seja ele uma div, uma imagem, um box, ou seja, qualquer elemento. body
Color Determina a cor do comando citado, baseado nas fontes RGB (Red,Green,Blue). head
Font-size Determina o tamanho em pontos (pt) do comando citado. head
Style Contém informações de estilo para um documento ou uma parte do documento. As informações de estilo específico estão contidas dentro deste elemento, geralmente no CSS head
